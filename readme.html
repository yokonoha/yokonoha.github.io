横茶横葉のサイト 利用規約

2025/10/20 改訂

本規約は、横茶横葉（以下、「管理者」といいます）が運営するウェブサイト（以下、「本サイト」といいます）および本サイトで提供されるすべてのコンテンツ、プログラム、サービス（以下、「本サービス」といいます）の利用条件を定めるものです。利用者は、本規約に同意した上で本サービスを利用するものとします。

第1条（総則および適用範囲）

  1. 本規約は、本サービスの利用者（以下、「利用者」といいます）と管理者との間の本サービスの利用に関わる一切の関係に適用されます。
  2. 利用者は、本サービスの利用をもって本規約に有効かつ取消不能な同意をしたものとみなされます。
  3. 管理者は、利用者の承諾なく本規約を随時変更できるものとします。規約変更後も利用者が本サービスを利用し続ける場合、当該変更に同意したものとみなします。
  4. 本サイトの特定の機能である「Contents Delivery」経由でのファイルダウンロードについては、付属のreadmeファイルが存在する場合に限り、readmeに記載された規約が適用され、本規約は適用対象外となります。これは著しく他者様の著作権が関係する成果物を配布する場合の特例的な利用規約免除措置です。
  5. また、本規約は当サイト(Contents Delivery機能を除く)や当サイトの利用規約の及ぶ以下のサイト「YOKONOHA(yokonoha.pages.dev)」に掲載されているコンテンツ・ 管理者(横茶横葉・みどりは)が制作し、SNS上で公開する一次創作作品を対象としています。二次創作作品(イラスト,立ち絵素材,動画などを含むがこれらに限らない)に対しては本規約は適用されません。

第2条（自己責任の原則および免責事項）

  1. 利用者は、本サービスを自己の責任において利用するものとし、本サービスの利用またはコンテンツのダウンロード・使用によって発生した一切の事象（損害、損失、金銭トラブル、機器の故障、ファイルの破損、インターネット接続料金などを含むがこれらに限らない）について、管理者は一切の責任を負わないものとします。
  2. 本サービスの利用により利用者間で、または利用者と第三者との間で発生した問題および争議は、当事者間で解決するものとし、管理者は一切関与せず、責任を負いません。
  3. 管理者は、本サービス上で提供するプログラム、ソフトウェア、アプリなどの動作、機能、正確性について、完全な保証をしないものとします（無保証）。利用者は、これらを自己責任で利用するものとします。
  4. 本サイトはセキュリティスキャン（VirusTotal）を行っていますが、第三者によるサーバー内ファイルの不正アクセスや改ざんが行われた場合、管理者は一切保証しません。また、アンチウイルスソフトによるウイルス検出に対する保証も行いません。
  5. 管理者は、本サイトおよび本サービスを予告なく変更、中断、または終了することができるものとします。これにより利用者に生じた損害について、管理者は一切責任を負いません。
  6. 管理者は、本サービスの維持、管理、存続に関して、その裁量により決定できるものとします。

第3条（知的財産権およびコンテンツの利用）

  1. 本サイトおよび本サービス上で提供されるコンテンツ（プログラム、ウェブページ、イラスト、アート、3D作品など）に関する著作権およびその他の知的財産権は、一部のアセットデータを除き、管理者に帰属します。
  2. 管理者が「著作権フリー」「フリー素材」「自由に利用可能」などの文言で二次利用を明示的に許諾しているコンテンツを除き、コンテンツの無断での二次利用、再配布、転載を禁止します。

  3. 利用者が許諾される行為

    • 本規約を遵守した上でのコンテンツの利用。
    • 本サイトURLのリンクシェア。
    • プログラムの不具合報告。
    • ダウンロードしたコンテンツの知人間での共有（コピーして渡す行為）。
    • 一般常識に従った利用。

  4. 二次利用に関する規定

    • コンテンツを二次利用または二次創作する場合、ソフトウェア・アプリの場合は「各種ライセンス」、イラスト・アート・3D作品の場合は「二次利用・二次創作の可否」や「商用利用の可否」などの明記された条件に従うものとします。
    • これらの条件が明記されていないコンテンツは、原則として二次利用はできません。二次利用を希望する場合は、事前に管理者に相談するものとします。
    • ダウンロードしたコンテンツは、私的目的の範囲内で利用するものとします（商用利用が可能と明記されたものを除く）。
    • 本サイトのプログラムや素材を活用して制作された成果物（作品）について、明示的な利用禁止事項がない限り、その成果物（作品）の著作権は制作者（利用者）に帰属します。

  5. 禁止される行為

    • 改変が可能と明記されていないコンテンツの改変。
    • リバースエンジニアリング（製作者から明確に許可された人物以外）。ただし、ソースコードが公開され、再利用可能な場合はこの限りではありません。
    • 著作権、商標権、その他の知的財産権を侵害する行為。
    • 明記がないにもかかわらず、本サイトのプログラムを他の場所で公開する行為。
    • 本サイトをインターネットアーカイブに手動で保存する行為。
    • 再利用が許可されている場合であっても、その成果物や派生作品そのものにおいて独占的な権利を主張する行為、または管理者や他利用者の迷惑や自由を侵害する行為。
    • 管理者のプログラムをそのまま、または改変して使用しているにもかかわらず、全て自作したと主張する行為。

第4条（禁止事項）

利用者は、本サービスの利用にあたり、以下の行為を行ってはならないものとします。

  1. 法令に反する行為、または犯罪行為に関連する行為。
  2. 公序良俗に反する行為。
  3. サーバーやネットワークの機能を破壊、または妨害する行為。
  4. 不正アクセス、または不正な目的での利用。
  5. 他の利用者、管理者、またはその他の第三者に不利益、損害、または不快感を与える行為。
  6. 本サービスおよびコンテンツを、政治的・宗教的なコンテンツの作成のために利用する行為。
  7. 人を欺くことや傷つけることを目的とした行為、または思いやりに欠けた行為。
  8. 本サイト以外で公開されている本サイトのアプリをダウンロードする行為（改ざんされている恐れがあるため）。
  9. 「著作権フリー」などと明記された素材であっても、悪意のあるプログラム（ウイルスソフト、トロイの木馬など）の製作に使用する行為。

第5条（プライバシー保護）

  1. 本サイトおよび管理者が運営する一部のサイト/アプリでは、ユーザビリティ・品質向上のため、またはアクセス数測定のために一般的な手法を用いたアナリティクスが一時的に使用されています。
  2. 利用者は、これに同意しない場合、アナリティクスをブロック可能なブラウザやブラウザ拡張機能等を利用するものとします。

第6条（料金および準拠法）

  1. 本サービスの利用料金は、原則として一切請求されません。万が一、利用料金の支払いを請求された場合は、専門の機関へ相談するものとします。
  2. 本規約の解釈にあたっては、日本国の法令を準拠法とします。
  3. 本サービスに関して紛争が生じた場合には、管理者の所在地を管轄する裁判所を専属的合意管轄裁判所とします。
