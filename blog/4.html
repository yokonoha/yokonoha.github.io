むかしのぱそこん
ジャンクT100TAを買って現世に蘇らせたよ!

こんにちは! 横茶横葉こと、横茶横葉です!

今回は初めてのブログ記事になります!

記念すべき第一回目は、ずばりジャンク! パソコンが好きな方の殆どはきっとジャンク好きですよね?（たぶん）私もその一人で、青い箱のあのお店には毎週のように通ってます。


ジャンク品のT100TA君。ヤスイ!!

さて、そんなその「青い箱が置いてあるお店」で2000円という価格でおいてあった2 in 1 PC,T100TA Transbookについて、現世に蘇らせるのにかなり苦戦したので書き残します!

T100TAについてはこの記事では端折りますが、atomなパソコンです。感の良い方は察したと思いますが、処理性能には全く持って期待できない what is called.... いにしへの機種 です。

win10なんてそんなステキなものは性能的に使えたものではありません。なので現代で実用機にするにはLinux化が必須とも言えそうです。

しかし、このPCのLinux化、一筋縄ではいかなかったです。


...RAMとストレージが...CPUが...

この機種、全ッ然性能的な余裕がありません。RAMは2GB,ストレージに関しては私の入手したものは32GB eMMCでした。 そしてatomというダブルパンチ... よよよ。これでwindowsはもちろん、Linuxでも重いディストロは使えないことがほぼ確定しました。

....で、でも、だ、大丈夫ですよ!Transbookくん、ぜひわたくしめにお任せください!現役復帰しましょうね!!(でっきるっかなあ...)

ということで、まず手始めに私の大好きな尊敬の念をも抱く超超軽量の万能Linux、antiX君をいれてみれば無問題! なーんて思っていたんですが.... antix君をインストールしてしばらくして気が付きました。

ネットブック時代からの宿命 Linuxで音が出ない!!

はい、これが最大の壁です。記憶が確かなら、これ、別の機種のN10Jc(GeForce 9300M GS搭載という少し異端な[ﾎﾒｺﾄﾊﾞﾀﾞﾖ]ネットブック)のときもこの問題に悩まされて、もう悩みたくないから売ってしまった って過去があったような...??

あぁ。そうでした。あのときから問題ありありなatom機は買わないようにしてたのに...忘れてました...!

でも今回はもう買ってしまったのでしょうがない! 責任感を持って現役復帰をさせていきましょう。

Transbook君、さあ、行きますよ!!!

ディストロ総当たり!!

というのも今回の音が出ない問題はalsaミキサーが正しいドライバーで動いていないというところにあります。antix君にドライバーを持ってってあげたり、色々コマンドで設定したりしたのですが、どうも動かないのです。

もちろんUSBDACを常備して外部出力におまかせするですとか、HDMI側の音声出力をセパレーターで拾って..なんて考えましたが、回りくどくて仕方ありません。

こうなったらもう新しいディストロ開拓へ船出するしかありません。

ということでやってみました....!

  • Linux Mint: 音出ずX
  • zorinOS: safemodeでしか起動せず 音はでたよ!! △
  • ChromeOS Flex: そもそもセットアップ画面にたどり着けず X
  • Spiral Linux Xfce: O ...!?音が出るよ?? こ、これだっ!!

Spiral Linux様 https://spirallinux.github.io/ Alsaドライバーがちゃんと当たってる
ボリューム調整の設定も可能に!

さて、spiral linuxという新たなディストロと出会い、alsa mixer君が正しいオーディオドライバーを見つけてくれたので、Transbook君は喋れるようになりました!!!

しかし、今度はまた問題が....


ん???? 勝手にスリープになる!!!

そうなのです。不可解なことにliveブート時もインストール時も一定間隔でなぜか勝手にtransbookくんが寝落ちしてしまいます。(起きて起きて-!)

どうやらスリープに使う磁気センサーまたは電源周りが暴走しているようです(詳しくは調べてないのでよく分かりませんでしたが)

このままでは使い物になりません。なので、live環境でインストールする際、そしてインストール後の初回起動時にこんなコマンドを頑張って打ち込みます。

sudo systemctl mask sleep.target suspend.target hibernate.target hybrid-sleep.target

すると、Transbook君はコーヒーを飲んだみたいになります(スリープしなくなる)。もっとちゃんとした説明をすると、シンボリックリンクを/dev/nullにすることで対処しています。(=つまりlid閉じてもスリープにはならない)


あとは、いつも通り、

sudo apt update

sudo apt install fcitx5-mozc fcitx5-config-qt fcitx5-frontend-gtk2 fcitx5-frontend-gtk3 fcitx5-frontend-qt5

sudo apt install im-config

とやって再起動し、

im-config -n fcitx5で日本語入力を有効化!

そしてaptやflatpakからいつものソフトをインストールするだけ!

やっと、やっとおわりました!!! これにて、Transbook君を現代でも通用するPCに生まれ変わらせました!!!!

所感



spiral linuxは初めて使いましたが、一言でいうと素晴らしいです!!! 私はXfce版を選んだのでなおさらですが軽い!動作が軽快!! またステキなOSに出会っちゃいました。Spiral君よろしくね!

そして、肝心のTransbook君ですが、なんとオンライン動画の1080p再生もブラウザで意外とこなしてしまうんです。私は驚きましたが、QSV搭載?なのかな?何かしらの再生支援が効いてる感じが少ししました。

そして、なんとびっくり、この記事も全部transbook君で書いてました!!!(FSWriteを使用: https://fswrite.pages.dev/) すごい....　私の買った個体はすごく電池持ちが良く6hぐらい持ってくれます。Micro-B充電なのがすこしあれですが、でも充電してしまえば長く使えますよ!!

Gpartedとか、ddコマンドをよく使うので、ずっと持ち運べるLinux機がほしかったのですがこれは嬉しい!しかもデタッチャブルでタブレットにもなっちゃうし、タッチまでできちゃう。つべも見れちゃうなんて...ステキ!

これからの相棒となること間違いなしです!よろしくね(相棒これで7台目だけど...ね。)


壁紙は新潟県村上市笹川流れ. いいところでしたよ!

皆様も中古で見かけたら、救ってあげてみてはいかがでしょうか?
